# Roadmap

- Complete the WASM crate by moving the browser renderer and `RustyMilkEngine` binding out of slskR.
- Publish `rustymilk-core` with stable parser/runtime APIs.
- Publish `rustymilk-wasm` build artifacts for browser clients.
- Convert `packages/rustymilk-web` into an installable package with typed exports.
- Update slskR to consume the standalone crates/package.
- Delete the old in-tree RustyMilk implementation from slskR after parity tests pass.
